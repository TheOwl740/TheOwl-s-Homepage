JavaScript Modules

I created these JavaScript modules for use in game development, but they might also be useful for other projects. The cards link to the .js file for the module.

Canvas

This module has a variety of functions suited for HTML canvas drawing. It automatically replaces the HTML document with a canvas that fills the current screen.

Input

This module detects user input in the form of button presses, mouse clicks, and mouse position.

Math

This module contains several mathematical functions, generally related to 2d space positioning for canvas games.

Timer

This module controls an update timer which fires the update() function repeatedly at a specified interval.

File Access

IN DEVELOPMENT
This module will allow you to easily update files with JavaScript.

Sound

This module is used to store and play sound effects.

The modules do not have documentation, however they are commented intuitively. Their basic functions should be easy to use and understand.
