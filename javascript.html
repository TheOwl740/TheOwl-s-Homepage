JavaScript Games

My first game development platform was JavaScript. I've made several games using the HTML canvas, and I now have a small library of modules for game development available on the JS Modules page. Though many of my projects were never completed, the playable and mostly playable ones will be included below.

Derp.io

This game is a knock off of the classic web game Diep.io. It is a top down combat game in which you level up a small tank and battle other tanks. It is only slightly unstable.

Chicken Jumper

This is the first game I ever developed, and is actually finished as well. It is based on the offline page dinosaur runner game by Google, but with a chicken.

Space Game

This generic named game is very generic itself. Unfinished, but probably stable, it is a space exploration game using the earliest incarnation of the canvas module.

Evolution Sim

This is not necessarily a game but more of a cool side project. It simulates a population of predators, prey, and plants, which inevitably falls apart. Press q to open stats page, e to speed up.
