Documentation

I've built my own personal documentation over time, which is meant to be easy to use with quick ctrl+f search. It has information on Javascript, CSS, HTML, C#, Bootstrap, and SQL. Click on any of the cards to be redirected to the documentation page for that language. From there you can navigate to other languages.

HTML

HTML (Hyper-Text Markup Language) is a coding language that defines the baseline structure of websites with a collection of nested elements.

JavaScript

JavaScript is a programming language primarily used to add interactivity to HTML sites.

CSS

CSS (Cascading Style Sheets) is used to style an HTML page by changing the layout and appearance with classes that apply to specified elements.

Bootstrap

Bootstrap is a library for CSS which provides classes to more effectively make a professional, scale responsive website.

SQL

SQL (Structured Query Language) is a programming language designed specifically for pulling, manipulating, and sorting information from a database.

C#

C# is a standard programming language with a wide variety of libraries and applications.
